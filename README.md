# Clarity

Clarity is a minimalist macOS cleaner that helps users reclaim disk space by manually selecting what to clean and by removing leftover files from uninstalled apps.

## What It Cleans
- User caches: ~/Library/Caches
- System logs: /Library/Logs and ~/Library/Logs
- Downloads folder: ~/Downloads (with confirmation)
- Xcode leftovers: ~/Library/Developer/Xcode/DerivedData
- Trash bin (permanent empty)

## Benefits
- Frees disk space with clear, manual control
- Reduces clutter from app leftovers
- Keeps the system tidy without background automation
